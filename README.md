# SkyCapital - Fast Business Funding Nationwide

SkyCapital is a trusted nationwide business funding provider serving all 50 states. We offer Merchant Cash Advance (MCA), working capital loans, small business loans, equipment financing, and revenue-based financing with 95% approval rate and funding in 24-48 hours.

## Features

- **Fast Approval**: Same-day approval with funding in 24-48 hours
- **High Approval Rate**: 95% approval rate for all credit types
- **Funding Options**: $5,000 to $5 million available
- **Nationwide Service**: Serving businesses in all 50 states

## Technology Stack

- Vite
- TypeScript
- React
- Tailwind CSS
- Supabase

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
